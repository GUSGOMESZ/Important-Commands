Here are some important commands related to Mix !

---

```bash
$ mix new project_name
```

Creates a new elixir project.

```bash
$ mix compile
```

Compiles the project.

```bash
$ mix deps.get
```

Fetches project dependencies.

```bash
$ mix test
```

Runs tests.

```bash
$ mix run
```

Executes the project.

```bash
$ mix format
```

Auto-formats code.
